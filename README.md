# Installation

```bash
go get -u github.com/greymd/ojichatrix
```

If `GO111MODULE=on` is set, follow below.

```bash
GO111MODULE=off go get -u github.com/greymd/ojichatrix
```

# Usage

Run the command.

```
$ ojichatrix
```

You'll see the awesome screen.
